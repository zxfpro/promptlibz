        assert template_type.value in [item.value for item in TemplateType]
        template = None

            template = """
            让我们一步步从用户的模糊需求中分析出明确且具体的需求步骤：

            第一步：提取用户表述中的关键词和重点内容。
            第二步：将关键词和重点内容转化成具体的步骤和需求。
            第三步：复核步骤和需求确保其明确具体且无二义性。
            第四步：将上述步骤和需求整理成明确的需求描述。

            用户需求表述：

            {user_statement}
